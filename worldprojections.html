Describe in your own words how you displayed the map in different projections using QGIS

In QGIS I added a layer using the shapefile in the world projections folder for the base map of the world. Then I used the indicatrix mapper plugin to add shapes to the map so that I could compare different projections' distortions. Then I opened the project properties and changed the coordinate reference systems for each world projection by searching the EPSG numbers.

Now, you should add the following projections on your own:

EPSG: 3857, 53018, 54034, 54027, 102016, and two additional projections that you choose.

EPSG 3857 Projection

EPSG 3857 is the Pseudo-Mercator projection is a variant of the Mercator projection but the computation is done on a sphere.

EPSG 53018 Projection

EPSG 53018 is the Winkel tripel projection is a compromise projection which minimally but generally distorts shape, area, distance, and direction.

EPSG 54034 Projection

EPSG 54034 is the cylindrical equal area projection which maintains relative area using a cylinder tangent.

EPSG 54027 Projection

EPSG 54027 is the World Equidistant Conic projection, a conic projection which preserves distances with equally spaced parallels.

EPSG 102016 Projection

EPSG 102016 is the North Pole azimuthal equidistant projection which preserves distance and direction from the central point.

EPSG 54030 Projection

EPSG 54030 is the World Robinson projection, a compromise projection which keeps all types of distortion relatively low, but distortion is not eliminated for any.

EPSG 26591 Projection

EPSG 26591 is the Monte Mario Italy zone 1 projection, a local projection using the Hayford ellipsoid.
